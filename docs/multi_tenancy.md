# Multi-tenancy (Концепция)

Приложение ProjectGT работает по модели **SaaS (Multi-tenant)**. 
Это означает, что данные каждой организации (Компании) полностью изолированы друг от друга.

## Основные правила:
1. **Компания — главный контейнер.** Всё (сотрудники, деньги, объекты) принадлежит компании.
2. **Переключение на лету.** Пользователь может состоять в нескольких компаниях и переключаться между ними через боковое меню.
3. **Безопасность.** Изоляция гарантируется на уровне базы данных (Supabase RLS). Если вы не состоите в компании, вы физически не сможете получить её данные.
