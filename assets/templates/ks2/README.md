# Шаблон КС-2 (`ks2_template.xlsx`)

Storage: `ks2_templates` / `ks2_template.xlsx`.  
Function: `export-ks2-form-header`.

## Подписи (эталон)

| Строка (базовый шаблон) | Содержимое |
|-------------------------|------------|
| **70** | Данные: **Сдал** + **Подрядчик:** + наименование как в шапке C11 (колонка **A**, высота **58**) |
| **71** | Линии подписи + подписи *должность* \| *подпись* \| *расшифровка* — только шаблон |
| 72–74 | Пустые |
| **75** | Данные: **Принял** + **Заказчик:** + наименование как в шапке C9 (колонка **A**, высота **58**) |
| **76** | Линии и подписи полей — только шаблон |

Должность и ФИО из договора **не подставляются**. «Генеральный директор» в шаблоне — статичный текст в файле, если нужен.

При расширении таблицы строки 70/75/71/76 сдвигаются на `rowShift + tableInsertShift`.

## Деплой

```bash
supabase functions deploy export-ks2-form-header
```
